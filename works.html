02 : Works

Here are some of the projects that I have worked on

001/006

Twitter Clone

Implemented a Twitter clone and client tester/simulator using Elixir and Actor Facility provided in Erlang. Used Phoenix web framework to implement a Web Socket interface. Implemented public key based AUTH method for same.

Discover

002/006

Review Rating Prediction Yelp Dataset Challenge

Used topic modeling and sentiment analysis to predict review ratings by building different prediction models using Latent Dirichlet Allocation for feature extraction with KNN, Multinomial Naive Bayes and Random Forest Algorithms.

Discover

003/006

Facial Recognition using Principal Component Analysis

Python implementation of Face Detection and Recognition system using Eigen Faces, Fisher Faces and Local Binary Pattern Histograms. Performed comparative analysis with the Linear Discriminant Analysis technique.

Discover

004/006

Huffman Encoding using D-ary Heaps

Developed encoder and decoder implementations for Huffman Encoding using Java framework. Instrumented code and performed experiments to determine bound on running times for Huffman tree generation using Binary Heap, Four Way Heap and Pairing Heap and used the fastest among them for Encoder implementation.

Discover

005/006

Bitcoin Mining System

The goal of this project was to use Elixir and the actor model to build a good solution to the bitcoin mining problem that runs well on multi-core machines. Our code was simultaneously run on 8 machines with one functioning as server and the other 7 as workers, and we were able to mine a coin with 7 leading zeroes. Technologies used: Erlang, Elixir

Discover

006/006

Pastry Protocol

We developed a distributed application in this project that built an overlay network using Pastry APIs with custom implementation for adaptive node join and requests routing. Through this we demonstrated scalable, decentralized object location in peer-to-peer systems which could consist of as many as 65,536 nodes. Technologies used: Erlang, Elixir
